# Subway Runner

A 3D endless runner game inspired by Subway Surfers, built with Three.js.

## How to Play

Open `index.html` in a browser.

### Controls

- **← → / A D** — Switch lanes
- **↑ / W / Space** — Jump over low barriers
- **↓ / S** — Roll under overhead barriers
- **Swipe** — Mobile touch controls

## Features

- 3-lane runner with smooth lane transitions
- Three obstacle types: tall trains (dodge), low barriers (jump), overhead barriers (roll)
- Coin collection with spinning gold coins
- Progressive speed increase
- High score saved to localStorage
- 3D environment with buildings, tracks, and shadows
- Running animation on the player character
- Mobile-friendly with on-screen buttons and swipe support
